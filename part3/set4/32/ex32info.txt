Exercise 32: modify tuples

We implement a manual version of modifying tuples in a similar pattern to the
sildes from Advanced Templates 1, but also note that <tuple> comes with a
std::tuple_cat function that has all this capability already implemented. 

 Q  Show what you have to change to use a tuple that has a third field of 
    type std::string (and show the program's output after having changed it).

 A  The only thing we need to change for this string to be added is the type
    signature of the tuples. We show the functional version including a string
    in the included main.cc file. The outcome will, rather then 4 and 3 
    (2 + 2 and 2 + 1) be 6 and 4 (3 + 3 and 3 + 1).
