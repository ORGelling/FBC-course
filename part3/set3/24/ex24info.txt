Exercise 24: Generalize conversions of nrs. to text

We use recursion to bite digits off our number one by one, storing them in a
char parameter pack, and prepending the rightmost digit to it at each step.

    calling I2C<123>::s_ntbs:
    
    instantiates I2C<123>, 
        with <size_t Remaining = 123, char ...Chars = [empty]>
    
    instantiates Build<12, '3'>
        with <size_t Remaining = 12, char ...Chars = '3'>
        
    instantiates Build<1, '2', '3'>
        with <size_t Remaining = 1, char ...Chars = '2', '3'>
        
    instantiates Build<0, '1', '2', '3'>
        with <size_t Remaining = 0, char ...Chars = '1', '2', '3'>
    
    which then inherits Digits, which converts all the chars from the
    parameter pack into an ntbs using:
        
        s_ntbs[] = { Chars..., 0 };
    
    which in our case gives:
    
        s_ntbs[] = { '1', '2', '3', 0 };
    
    *This* is also the ::s_ntbs that is called from the initial call to:
        
        Build<12345>::s_ntbs
